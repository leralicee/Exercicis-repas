Cálculo de Área y Perímetro de Figuras

Fes un programa que calculi l'àrea i el perímetre de diferents figures geomètriques. L'usuari haurà d'escollir la figura i introduir les dades necessàries.

Output esperat:

=== CALCULADORA DE FIGURES ===
Escull una figura:
1. Quadrat
2. Rectangle
3. Cercle
4. Triangle
Opció: 2
Introdueix la base: 5
Introdueix l'altura: 3
--- RESULTATS ---
Perímetre: 16.0
Àrea: 15.0
